# 🎸 Revival Band

> *A digital songbook of Tamil Christian worship chord sheets — organized, accessible, and always up to date.*

---

## 📖 About Us

**Revival Band** is a worship team passionate about leading praise and worship through Tamil Christian songs. We believe music is a powerful way to glorify God, and this repository is our **shared songbook** — a place where every band member can pull up chord sheets and setlists instantly, whether it's during rehearsal or right before a Sunday service.

All our songs are written in Tamil with chords placed directly above the lyrics, so you can read and play along at a glance — on your phone, tablet, or laptop.

---

## 📂 How This Repo Is Organized

```
Revival-Band/
├── Songs/                                      # All songs, grouped A–Z
│   ├── K/  → Kerubin Serabinghal.md
│   ├── M/  → Magimayin Raja Maghimaiyodu.md
│   └── Y/  → Yesu Ennaku Jeevan.md
├── Sunday Sets/                                # Weekly setlists by date
│   └── 08-03-2026.md
└── Resources/                                  # Guides & references (coming soon)
```

- **`Songs/`** — The master library. Each song has its own file, grouped into folders by first letter for quick lookup.
- **`Sunday Sets/`** — Complete chord booklets for each Sunday service. One file = all the songs for that day.
- **`Resources/`** — A space for music guides, scale charts, and references *(coming soon)*.

## 📝 How to Read the Chord Sheets

Every song uses a simple **chord-over-lyrics** format — chords sit right above the word where the change happens:

```text
F#                    B
மகிமையின் ராஜா மகிமையோடு
C#                 F#
வருகின்றார் மேகமீதில்
```

### Sections You'll See

| Label | What It Means |
|-------|--------------|
| **INTRO** | Instrumental opening — just chords, no lyrics |
| **Verse** | Main verse with chords above the lyrics |
| **Pre-Chorus** | A build-up section before the chorus |
| **Chorus** | The main refrain / hook |
| **Stanza** | Additional verses or bridge sections |
