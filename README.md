# 🎸 Revival Band

A version-controlled worship songbook for **Revival Band** — a curated collection of Tamil Christian worship song chord sheets, organized by weekly Sunday setlists.

---

## 📖 About

Revival Band is a worship team dedicated to leading praise and worship through Tamil Christian songs. This repository serves as our **digital songbook** — a shared, version-controlled resource where band members can access chord charts, setlists, and musical resources anytime, anywhere.

### Why Git?

- 🔄 **Version History** — Track every change to chords, keys, and lyrics
- 👥 **Collaboration** — Band members can suggest edits and additions
- 📱 **Accessibility** — Access setlists from any device via GitHub
- 📦 **Organized** — Clean folder structure keeps everything easy to find

---

## 📂 Repository Structure

```
Revival-Band-/
├── Songs/               # 🎵 Master song library — individual chord sheets
├── Sunday Sets/         # 📅 Weekly setlists organized by date
│   └── DD-MM-YYYY.md    # Each Sunday's song set with chords & lyrics
├── Resources/           # 📚 Band resources — guides, scales, tutorials
├── LICENSE              # MIT License
└── README.md            # You are here
```

| Folder | What's Inside |
|--------|---------------|
| **`Songs/`** | Individual song files — one `.md` per song with full chord charts |
| **`Sunday Sets/`** | Weekly setlists — each file contains the songs for a specific Sunday service |
| **`Resources/`** | Helpful resources — music theory, scale charts, gear guides, tutorials |

## 📝 Chord Sheet Format

Each song follows a consistent **chord-over-lyrics** format:

```
Chord1                        Chord2
தமிழ் பாடல் வரிகள் (Tamil lyrics here)
Chord3                        Chord4
அடுத்த வரி (Next line of lyrics)
```

### Song Structure

Every chord sheet includes the following sections where applicable:

- **INTRO** — Instrumental intro chords
- **Verse** — Main verse with chords
- **Chorus** — Chorus / refrain section
- **Stanza** — Additional verses or bridge sections

## 🤝 Contributing

Band members and contributors are welcome to help grow the songbook!

### Adding a New Song

1. Create a new `.md` file in the `Songs/` folder
2. Name it using the transliterated song title (e.g., `kerubin-serabinghal.md`)
3. Follow the chord sheet format described above
4. Include: **Song title** (Tamil + transliterated), **Key**, **Tempo**, and all sections

### Adding a Sunday Setlist

1. Create a new `.md` file in the `Sunday Sets/` folder
2. Name it using the date format: `DD-MM-YYYY.md`
3. List all songs for that Sunday with full chord charts
4. Include a **Table of Contents** at the top for easy navigation

### Suggesting Changes

- **Key change?** Update the chords and mention the new key in the header
- **Lyric correction?** Edit the Tamil text directly
- **New arrangement?** Add or modify sections as needed
